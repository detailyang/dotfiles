# Word Mode

适用场景：用户要求解释、剖析或深入理解一个英文单词。

这个模式不做普通翻译，目标是让用户抓住这个词的核心语义、词源画面和现代语感。

## 适用边界

- 只处理单个英文单词
- 不用于中文概念
- 不默认用于英文多词短语，除非用户明确要求分析这个词组

## 执行步骤

### 1. 标题行

输出格式：

```markdown
## {Word} /{IPA}/ {中文翻译}
```

如果音标不确定，就省略，不要编造。

### 2. 核心语义

输出三个部分：

- **原始画面**
  - 用一句话描述这个词最源头、最物理的画面
- **核心意象**
  - 压成一个短公式
  - 例如：`warmth + time + protection = incubation`
- **解释**
  - 用有穿透力的语言把词源、现代用法和语感串起来
  - 重点解释它为什么不是某些近义词

### 3. 一语道破

用一句中英双语金句总结这个词的灵魂，使用引用格式：

```markdown
> "English sentence. 中文金句。"
```

## 输出标准

- 重点是掌握词的灵魂，不是列词典义项
- 解释要把抽象含义长回具体画面
- 金句要能脱离上下文单独成立
