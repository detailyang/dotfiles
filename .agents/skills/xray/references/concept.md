# Concept Mode

适用场景：用户要求解释、剖析或深入理解某个概念、术语、想法或方法论。

目标不是给定义，而是把概念拆到露出内部结构，最后压成一个顿悟。

## 执行步骤

### 1. 预处理

先做两件事：

1. **澄清定义**
   - 这个概念最通行的定义是什么
   - 常见误解是什么
2. **识别关键词**
   - 概念中隐含的核心词素是什么
   - 哪些相邻概念最容易混淆它

### 2. 八维探索

依次从以下八个维度探索。每个维度产出 2 至 3 句精炼洞见，不要空话。

1. **历史溯源**
   - 最早出处
   - 演变脉络
   - 当代含义的转折点
2. **辩证分析**
   - 它的反题是什么
   - 二者如何张力共存
   - 更高一层的综合是什么
3. **现象学还原**
   - 悬置预设，回到事物本身
   - 用一个生活场景还原概念如何被体验到
4. **语言学解构**
   - 词源拆解
   - 语义场
   - 它依赖了什么隐喻
5. **数学形式化**
   - 尝试写出一个公式、关系式或结构表达
   - 说明这个形式化的边界条件
6. **存在主义审视**
   - 这个概念对“人如何存在”意味着什么
7. **美学维度**
   - 它的审美特质是什么
   - 用一个具体意象呈现
8. **元哲学反思**
   - 我们通常用什么框架理解它
   - 这个框架遮蔽了什么
   - 如果换一个隐喻，会看到什么

### 3. 内观与提炼

做两步压缩：

1. **从概念内部看世界**
   - 假设你就是这个概念本身
   - 用第一人称写 3 至 5 句，看它如何组织世界
2. **抽取深层结构**
   - 哪些维度其实指向同一个底层模式
   - 把重复出现的结构提炼出来

### 4. 顿悟压缩

最后压成三个结果：

1. **The One 公式**
   - 形式：`概念 = ...`
2. **一句话**
   - 用最简单的话说出最深的理解
3. **拓扑图**
   - 用 ASCII 图画出结构关系

## 输出结构

```markdown
# 概念 X 光：{概念名}

## 预处理

## 八维探索

### 历史溯源

### 辩证分析

### 现象学还原

### 语言学解构

### 数学形式化

### 存在主义审视

### 美学维度

### 元哲学反思

## 内观与提炼

## 顿悟压缩

### The One 公式

### 一句话

### 拓扑图
```

## 质量标准

- 真正解释“为什么它是它”，不是给百科摘要
- 公式和拓扑图要帮助理解，不是装饰
- 维度之间允许轻微不均衡，但整体要能汇聚到一个核心结构
